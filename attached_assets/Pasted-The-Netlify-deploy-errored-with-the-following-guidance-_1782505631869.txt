The Netlify deploy errored, with the following guidance provided:

**Diagnosis**
- The build runs `npm run build`, but `[line 63](#L63)` shows `sh: 1: next: not found`, so the Next.js CLI isnвЂ™t present in `node_modules`.
- In [`package.json`](https://github.com/nathanaelLarkum/NJLarks/blob/main/package.json), `next` is listed under `devDependencies`. Because Netlify builds run with `NODE_ENV=production`, `npm ci` skips dev dependencies, leaving the `next` binary unavailable.

**Solution**
Move `next` (and any other build-time requirements) from `devDependencies` into `dependencies`, then commit the change:

```json
// package.json
{
  "dependencies": {
    "next": "15.5.19",
    "react": "...",
    "react-dom": "..."
  },
  "devDependencies": {
    // keep only true dev tools here
  }
}
```

After updating, push the commit so Netlify installs Next.js during the build and rerun the deployment.
