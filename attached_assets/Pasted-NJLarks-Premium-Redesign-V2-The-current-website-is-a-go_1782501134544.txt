NJLarks Premium Redesign V2

The current website is a good MVP but it does not yet feel like a premium software consultancy.

This redesign should focus on:

* Better visual hierarchy
* Better typography
* Better storytelling
* Better project presentation
* Better transitions
* Better imagery
* Better mobile experience
* Higher perceived value

The goal is that visitors believe NJLarks is capable of delivering £10,000+ software projects while still offering affordable services.

Motion & Animation

Current animations are too basic.

Create premium microinteractions.

Use Framer Motion.

Requirements:

* Smooth fade reveals
* Staggered card animations
* Hover elevation
* Subtle glow transitions
* Smooth navigation transitions
* Magnetic CTA buttons

Do NOT:

* Use parallax
* Use excessive movement
* Hurt performance

Animations should feel like Vercel or Linear.
